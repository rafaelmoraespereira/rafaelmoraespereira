📌 Objetivo do Projeto

Este projeto tem como objetivo que os alunos criem um portfólio online, apresentando de forma didática e acessível o funcionamento dos servidores e seus serviços essenciais. A ideia é publicar o conteúdo na internet, tornando-o parte do portfólio profissional, algo que poderá ser compartilhado futuramente com recrutadores ou em redes sociais técnicas.

📌 Formato e Ferramentas para Publicação

Os alunos deverão desenvolver o trabalho em duplas, utilizando plataformas digitais para publicar seu portfólio. Cada dupla poderá escolher um dos seguintes formatos:

1️⃣ Website ou Blog

📌 Criar uma página ou post explicativo em uma plataforma como:

  • Notion → Fácil de usar, permite inserir textos, imagens e links interativos.
  • Google Sites → Simples e gratuito para criar um site estruturado.
  • WordPress / Blogger → Para quem deseja criar um blog mais profissional.
  • GitHub Pages → Para duplas que tenham conhecimento básico de HTML/CSS.

2️⃣ Publicação Técnica no LinkedIn ou Blog Técnico

📌 Criar um post estruturado e didático, explicando cada parte do conteúdo.

  • O LinkedIn pode ser usado para um artigo técnico sobre os servidores.
  • O Medium, Dev.to ou Hashnode são opções para um blog técnico.
  • O conteúdo deve ser bem estruturado e utilizar imagens ou diagramas.

3️⃣ Vídeo Explicativo no YouTube ou Instagram Reels

📌 Criar um vídeo curto (de 3 a 5 min) explicando os conceitos aprendidos.

  • Pode ser um vídeo tradicional, com explicação verbal e slides.
  • Pode ser um vídeo animado usando ferramentas como Canva, Powtoon ou Animaker.

📢 Independente do formato escolhido, o link do trabalho publicado deve ser enviado para avaliação.

📌 Conteúdo do Portfólio

Cada dupla deverá criar e publicar um material digital abordando os seguintes tópicos:

1️⃣ O que é um Servidor?

  • Definição e função de um servidor.
  • Diferença entre servidores dedicados e compartilhados.
  • Exemplos reais: Empresas que dependem de servidores para rodar aplicações.

2️⃣ Diferença entre Servidores Físicos e Virtuais

  • Explicação sobre servidores físicos e servidores em nuvem.
  • Benefícios e desafios de cada um.
  • Comparação entre hospedagem local vs. cloud computing.

3️⃣ Escolha 1 serviço de rede para aprofundar

📌 Cada dupla deverá escolher APENAS UM dos seguintes serviços e explicá-lo em detalhes:

  • Servidor DHCP – Como funciona a atribuição automática de IPs.
  • Servidor DNS – Como traduz endereços de sites para IPs.
  • Servidor Web (HTTP/HTTPS) – Hospedagem de sites e segurança na web.
  • Servidor de Arquivos – Como funciona o compartilhamento de arquivos em redes.

📌 Para o serviço escolhido, os alunos devem incluir:

  • Definição e funcionamento.
  • Importância para redes locais e para a Internet.
  • Exemplos reais e aplicações no dia a dia.
  • Prints de telas, diagramas ou fluxogramas (se possível).

4️⃣ Parte Prática (Opcional, mas recomendado)

📌 Cada dupla pode adicionar uma experiência prática, como:

  • Testar um servidor DHCP ou DNS em uma VM e mostrar prints.
  • Criar uma configuração básica de servidor Web local.
  • Fazer um mini-tutorial com comandos básicos usados para administrar servidores.

5️⃣ Conclusão e Reflexão Final

  • Resumo do que aprenderam e como isso pode ser útil na carreira.
  • Como esse conhecimento se aplica a um futuro trabalho na área de redes.

💡 Dicas para um Bom Portfólio Digital

  • Escolha a plataforma que melhor se encaixa no seu objetivo – Google Sites, LinkedIn, Medium, Notion, WordPress, YouTube.
  • Use imagens e diagramas – Torne o conteúdo mais visual e didático.
  • Crie um conteúdo acessível e compartilhável – Algo que possa ser lido por outras pessoas interessadas no tema.
  • Demonstre conhecimento técnico – Inclua prints de ferramentas, exemplos práticos ou comandos básicos.
  • Faça algo que vocês possam mostrar no futuro – O objetivo é ter um material publicável e útil no portfólio profissional.
